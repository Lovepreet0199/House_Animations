Welcome to The Snap Room

The Snap Room is an interactive space where animations bring your imagination to life. You can freeze moments, race the cats, and explore a dynamic, animated environment created entirely with HTML, CSS and minimal JavaScript.

About the Creator

Hi, I'm Lovepreet Sandhu, a web enthusiast about css animations and interactive design. This project showcases creative use of 3D transforms, keyframe animations, and accessibility best practices.

Vision & Inspiration

The goal of this project was to combine a playful, animated experience with practical web development skills. Each element is designed to be visually appealing, accessible, and interactive
